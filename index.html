Fast

Mue Tab utilises many technologies, to increase it's efficiency and overall speed and smoothness for the user, while remaining non-intrusive and with low resource usage.

Transparent

At Mue, transparency is key. We strive for complete end to end transparency, we collect no personal data and if you would like to check yourself, all the code is open source on GitHub.

Customisable

Mue has settings inside the new tab, which allow it to be customised such as background blur and toggling whether elements should appear or not. All of this is locally stored and not shared with a server.
